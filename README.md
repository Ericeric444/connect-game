# 🎮 连连看游戏 (Connect Game)

一个用 HTML、CSS 和 JavaScript 开发的在线连连看游戏，可直接在浏览器中打开并游玩。

## ✨ 游戏特性

- 🎯 **8×4 游戏网格** - 32张卡片，16对表情符号
- 🎮 **简单易玩** - 点击卡片配对相同的表情
- 📊 **实时统计** - 显示分数、剩余对数和游戏时间
- 💡 **提示功能** - 显示一张未匹配的卡片（2秒）
- ↩️ **撤销功能** - 撤销已配对的卡片
- ⏸️ **暂停功能** - 随时暂停和继续游戏
- 📱 **响应式设计** - 完美支持手机、平板和桌面设备
- ✨ **流畅动画** - 平滑的翻转和匹配动画

## 🚀 快速开始

### 方式一：直接打开（最简单）
1. 下载或克隆本仓库
2. 用浏览器直接打开 `index.html` 文件

### 方式二：使用本地服务器

**Python 3:**
```bash
python -m http.server 8000
```

**Node.js:**
```bash
npx http-server
```

然后访问 `http://localhost:8000`

## 📁 文件说明

- **index.html** - 游戏主页面
- **style.css** - 游戏样式和动画
- **game.js** - 游戏逻辑

## 🎮 游戏规则

1. 点击卡片翻开，寻找相同的表情符号
2. **配对成功** ✅ - 得到 100 分
3. **配对失败** ❌ - 扣 10 分
4. 用最少的时间和最高的分数完成所有配对
5. 完成所有 16 对时游戏结束

## 🕹️ 游戏按钮说明

| 按钮 | 功能 |
|------|------|
| **新游戏** | 开始新的游戏轮次，重置分数和计时器 |
| **提示** | 显示一张未匹配的卡片（2秒后隐藏） |
| **撤销** | 撤销上一次配对（只能撤销已配对的卡片） |
| **暂停** | 暂停游戏计时和操作，点击继续 |

## 🎨 游戏特色

- 紫色渐变背景，现代简洁界面
- 流畅的卡片翻转动画
- 成功配对时的绿色提示

## 📊 评分系统

- 配对成功：+100 分
- 配对失败：-10 分（最低不低于 0 分）
- 分数越高越好！

2. 利用提示功能来了解卡片位置

4. 先完成你有把握的配对
